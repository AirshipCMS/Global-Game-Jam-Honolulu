Upcoming Workshops

Rapid Prototyping & 2D Game Dev with Unity

This workshop hosted by DevLeague will be held on Thursday, January 19, from 6pm - 8pm in the Manoa Innovation Center Ideation Room.

- Learn how to use the Asset store to find premade assets and import them into your project for quick prototyping.
- Learn about Unity 2D game development and 2D game collisions.
- Learn about particle effects.

Web-Based Deployment

On Saturday, Jan 21, from 10am - 10:45am, will be demonstrating a rapid development & deployment workflow using AirshipCMS.io, our software sponsor's product. All attendees will receive a free account to deploy their hackathon projects. Accounts include web & asset hosting.
